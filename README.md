# film-streaming

+ Tạo file .env:
  - Example:  
  ```
  PORT=9096
  HTTP_CORS=*

  DB_HOST=localhost
  DB_PORT=3306
  DB_NAME=film_streaming
  DB_USER=root
  DB_PASSWORD=
  CORS_URL=['*']

  MAIL_ADDRESS=
  MAIL_ENC=
  MAIL_PASSWORD=
  MAIL_NAME=
  ```

+ Chạy api:
  - cd api
  - npm install
  - npm run dev
